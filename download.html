Download

Novaroma is free open source software. You can search and download tv shows, movies and subtitles automatically.

Also you can use Novaroma with dlna media servers (XMBC, Kodi, Plex, Serviio, PS3 etc.)


*Novaroma uses μTorrent/BitTorent or Transmission for downloading. If μTorrent/BitTorent or Transmission is not installed on your computer, Novaroma will run a portable version of μTorrent.



Source Code

If you want to download or have a look at the source code, you can do so at Novaroma Github respository
